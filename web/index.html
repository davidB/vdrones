About VDrones

In 'VDrones', player drives a vdrone across areas to grab cubes.

'VDrones' is a 3D game for modern web browser (Windows, Linux, Mac). It follows the web standard : update often, no plugins.

Only the firsts areas are playable for free. The game is a 'Pay to play fully' (not a 'Free To Play'/'Pay to Win')

Future

  • More FUN, better areas (graphics, sounds, effects, design), more areas, more enemies, more traps, more defenses...
  • A garage and a shop to tune vdrone
  • Gamepad support
  • Play on tablets (Android, iOS)
  • Guide to create area
  • Multi-player modes
  • ...

Suggestions and feedbacks are welcome, use The Google+ community page.

Technologies

  • The client side is open source (CC-BY-SA 3.0)
  • Languages: Dart/Javascript, html5, Stylus/CSS3, GLSL, Go
  • Hosting: Google App Engine, Google+, Google Play Game Sevices

Full list of technologies used to create and to run the site is available into Humans.txt
